shila-css

Sass-based CSS framework.

This documentation is for shila-css 1.0.0.

Settings

Global variables that are used to configure the framework and other useful variables for common use cases.

Tools

Global functions and mixins.

Generic

Low-specificity, far-reaching rulesets (e.g. resets).

Elements

Useful default rules for unclassed HTML elements.

Styles

The opinionated part of the framework. Some light styling to make unclassed HTML elements look and feel a bit nicer. Heavily based on Bitters.
